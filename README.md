# SearchEngine-game
Java实现的一个简单的本地的搜索引擎，希望可以帮助到你

### 第一步，实现网络爬虫
通过运行WebClawer.java文件，设置相应的网站，爬取对应的信息，记住一定是需要用来构建索引的信息，是可以帮助搜索的可取的信息功能。

### 第二步，构建索引
通过运行IndexBuilder.java文件，构建索引，具体代码如文件所示，记得修改相对应的信息

### 第三步，实现简单的图形搜索界面
通过运行SearchEngine-GUI.java文件，一个简单的图形界面可以实现简单的搜索功能
